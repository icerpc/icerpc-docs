    ResettablePipeReaderDecorator(PipeReader, int)

    Constructs a resettable pipe reader decorator. This decorator avoids consuming the read data to allow restart reading from the beginning. The decorator becomes non-resettable and IsResettable returns false if the buffered data exceeds the maxBufferSize, if the reading fails or if it's canceled with CancelPendingRead().
