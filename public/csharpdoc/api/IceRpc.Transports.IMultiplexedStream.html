    ReadsClosed

    Gets a task that completes when all read network activity ceases for this stream. This occurs when:

    • System.IO.Pipelines.PipeReader.Complete(System.Exception) is called on this stream's System.IO.Pipelines.IDuplexPipe.Input.
    • the implementation detects that the peer wrote an "end stream" to mark a successful write completion.
    • the peer aborts writes by calling System.IO.Pipelines.PipeWriter.Complete(System.Exception) with a non-null exception on the stream's System.IO.Pipelines.IDuplexPipe.Output.
    • the implementation detects a network failure that prevents further reads on the underlying network stream.
    The task is never faulted or canceled.
