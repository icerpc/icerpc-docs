    Interface IProcessService

    Interface used to implement services for Slice interface Process. IProcess. A server application managed by a locator implementation such as IceGrid hosts a Process service and registers a proxy to this service with the locator registry. See SetServerProcessProxyAsync(string, ProcessProxy, IFeatureCollection?, CancellationToken).
