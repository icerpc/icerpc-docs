---
title: Mapping for basic types
description: Learn how basic Ice types are mapped to C#
---

The Ice built-in types are mapped to C# types as shown below:

| Ice    | C#     |
|--------|--------|
| bool   | bool   |
| byte   | byte   |
| short  | short  |
| int    | int    |
| long   | long   |
| float  | float  |
| double | double |
| string | string |

The mapped C# type has always exactly the same name as the Ice type.
